ENTRAÍ

O hub dos grandes eventos.

QR Code Generator

A Entraí é uma plataforma de ticketing robusta, projetada sob o paradigma de sistemas resilientes e escaláveis. O projeto foca em resolver o desafio clássico de concorrência em inventários finitos e processamento de dados em tempo real.

Tipografia

Abcdefghijklmnopq.

Colorimetria


Escopo do Projeto

  • Portal do Organizador: Dashboard para criar eventos, definir lotes de ingressos (ex: VIP, Meia, Inteira) e visualizar gráficos de vendas em tempo real.
  • Experiência do Usuário: Busca por geolocalização, filtros por categoria e checkout fluido.
  • Validação Real-time: Geração de QR Codes únicos para cada ingresso e uma interface simples de "check-in" para o organizador validar na entrada.

Diferenciais

    Integração com Calendários: Implementar o envio de arquivos .ics por e-mail e integração via API para sincronização automática.
    Webhooks & Notificações: Sistema de e-mails transacionais (via SendGrid ou Mailgun) confirmando a compra e enviando o PDF do ingresso.
    Modo Offline (PWA): O organizador pode precisar validar ingressos em um local com internet instável.
